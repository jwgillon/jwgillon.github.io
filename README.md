# 🚀 Instant PowerPoint Creator
**Powered by Gemini AI**

An automated educational tool designed to transform simple themes into fully playable, classroom-ready PowerPoint games in seconds. Specifically built for ESL educators.

## 🎮 Featured Games

* **Jeopardy:** A 5-category board with 25 custom clues, answers, Final Jeopardy, and a live 4-team scoreboard.
* **Family Fjord:** 10 survey-style questions with ranked answers and point values.

## 🛠️ How it Works

1. **Enter a Theme:** Provide a topic (e.g., "Marine Biology in Huatulco").
2. **AI Processing:** Gemini generates relevant, level-appropriate questions.
3. **Instant Download:** The system compiles a custom-styled .pptx file.

## 💻 Tech Stack

* **Backend:** Flask (Python) hosted on Render.
* **AI:** Google Gemini API.
* **Frontend:** HTML5, CSS3.
